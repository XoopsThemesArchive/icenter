This is XOOPS theme based on CSS template ( www.solucija.com ).

There are marquee in header display what you put in intro.html , and two links for adding your site to favorites ans set as homepage.

<{$xoops_sitename}>,<{$xoops_slogan}> , <{$xoops_meta_description}> and <{$xoops_footer}> inserted in theme.html so you need to set up it from admin panel > General Settings and Meta Tags and Footer .

Using clblocks and crblocks not recommened.
 
Enjoy !
